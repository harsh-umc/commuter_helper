# commuter_helper
A prototype mobile application to help students without personal vehicles coordinate rides in small-town campus environments.

Commuter Helper is a prototype mobile-based application designed to help students without personal vehicles connect with students who are willing to provide rides. The application focuses on safety, usability, and community trust, especially in small-town campus environments.

## Project Purpose
This project is developed as part of a software engineering course to demonstrate requirements gathering, system architecture design, project planning, and testing strategies.

## How to Run
1. Install Python 3
2. Run(Terminal): python commuter_helper.py

## Data is Stored in commuter_data.json

## Features (Prototype)
- User registration for drivers and commuters
- Profile creation with basic information
- Ride request posting and acceptance
- Community-style feed for ride requests
- In-app communication between users
- Review and rating system for drivers
- Planned university authentication (prototype)
